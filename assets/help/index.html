This program creates VEOs out of one or more directories in a filesystem

Main pane

The main controls are located on this pane. The controls are as follows:

  • Directories to Harvest This is an area where the directories to capture are entered. Each directory entered will result in a single VEO. The VEO will contain the contents of that directory, and the contents of the subordinate directories. The 'Browse' button on the right is used to add directories, and the 'Delete' button to remove them.
  • Template Directory This field specifies the directory that contains the templates used to construct the VEO. Normally, a standard template directory should be used. If necessary, the 'Browse' button on the right can be used to select a new directory.
  • Output Directory This field specifies the directory where the VEOs are to be created. The 'Browse' button on the right can be used to select a new directory.
  • PFX file This field specifies the PFX file used to sign the VEO. The 'Browse' button on the right can be used to select a new directory.
  • PFX Password This field password for the PFX file used to sign the VEO.

Exc Folders pane

This pane is used to select directories and/or files that should not be harvested. The 'Browse' button on the right is used to add directories or files, and the 'Delete' button to remove them

Exc Files pane

This pane is used to select patterns directories and/or files that should not be harvested. For example, the pattern '*.tmp' will exclude any files that have a name ending in '.tmp'. There is one pattern per line.

Advanced pane

This pane is used to control advanced functionality; in particular the level of logging and the hash algorithm to be used.
